# MarketVibe homepage repositioning preview

This document records the approved direction for the homepage refresh.

Do not position MarketVibe as a Facebook scraper or scanner.

Customer-facing positioning:

- Opportunity intelligence for modern prospecting
- Qualified opportunities before competitors find them
- Lead opportunity scoring
- Audit summaries
- Outreach angle suggestions
- Faster prospect discovery

Do not change checkout, pricing, Stripe, auth, API routes, database schema, or plan logic.
